##Template Overrides Kategorie:

###wbcblogfields

Gibt in einem Kategorieblog die Customfields als Tab/Accordion aus. Die Variante lässt sich in den Parametern einstellen. Standard ist Accordion. Welche Custom Fields als Accordion / Tab ausgegeben werden sollen, kann man in den Parametern auswählen. Die Custom Fields müssen mit dem Wert “Tabs” im Feld “Note” im jeweiligen Customfield gekennzeichnet werden.

Die Accordions / Tabs mit den Feldern werden unterhalb des Beitrag ausgegeben. **zusätzliche Optionen Layout: Akkordeon oder Tabs Erstes Element offen Custonfields die in den Tabs erscheinen sollen

**zusätzliche Optionen: Einleitungstext kürzen Weiterlesen für Leading Beiträge Leadingbeiträge kürzen Schlagworte verlinken

###wbcblogaccordion

Gibt die Einleitungs Beiträge einer Kategorie als Accordions aus. Über die Parameter kann man einstellen, ob der erste Betrag aufgeklappt sein soll. **zusätzliche Optionen Layout: Erstes Element offen

**zusätzliche Optionen: Akkordeon / Tab Titel Schlagworte verlinken
